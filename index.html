Supported Board

Installation Steps
  1. Connect the XIAO ESP32-S3 Sense board using USB cable
  2. Open this page in Google Chrome or Microsoft Edge
  3. Click the Connect button below
  4. Select the USB Serial Port
  5. Wait for firmware flashing to complete
